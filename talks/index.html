Talks

Reflection (slides)

Tour of reflection, the good, the expensive, and the insecure. This talk is based on my experience while working on scylladb/gocqlx, an ORM for Scylla (and Cassandra). I also show 80-90% performance gain using “generics” (2019) in scylladb/go-set.

Adventures with SSH (slides)

I present how to use SSH daemon as HTTP reverse proxy, and SSH as HTTP transport. This setup was used in Scylla Manager 1.x and was removed in Scylla Manger 2.x as we needed to run code in Scylla nodes. After that we open-sourced scylladb/go-sshtools, a wrapper around SSH client.

HTTP Tunnels to localhost (YouTube)

I present mmatczuk/go-http-tunnel, HTTP/2 based ngrok clone. The ideas here are based on my prior work on koding/tunnel. Now the project is little abandoned.
