Projects

mmatczuk/scylla-go-driver

Experimental, high performance, ScyllaDB driver. I’m leading a group of four University of Warsaw (mimuw) students. The work is their BS thesis. The goal is to implement a lock-free driver with smart memory usage (GC sympathy). Initial results look very promising.

mmatczuk/go-http-tunnel

Ngrok clone implemented with HTTP/2 streams.

HTTP tunnels are also great for exposing applications running locally to the outside world; even if you don't have a public IP. @michalmatczuk makes this easy for us with his super dope go-http-tunnel project: https://t.co/XCmk9cwBH0 https://t.co/BdFY4TqDMp

scylladb/scylla-manager

The ScyllaDB cluster backup and repair solution used in virtually every ScyllaDB cluster! See the product website and the docs. It saves thousands of $ by deduplicating the sstable files. It uses a novel method of parallel repairs, described in this post. As part of this project, my team and I, sent many patches to rclone:

  • Adding context.Context to the whole project
  • Adding fadvise to manage Linux page cache while coping TiB of files
  • Multiple bugfixes to s3 backend implementation

scylladb/gocqlx

CQL query builder, reflection based ORM, and migration tool. I started this project when I joined ScyllaDB in 2017. It is used internally by Scylla Manager, externally by many ScyllaDB users, and over 60 projects on GitHub. There are over 600 daily clones and 100-200 page views on GitHub.

scylladb/go-set

Pre Go generics type-safe, zero-allocation port of fatih/set package. This project uses mmatczuk/go_generics a bazel-free version of Google go_generics tool released with gVisor project.
